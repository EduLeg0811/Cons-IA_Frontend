# Repository Editing Rules

- All text source files in this repository must be saved as UTF-8 without BOM.
- Prefer normal Unicode characters in source code when the file already contains them.
- Do not replace accented Portuguese text with HTML entities or JavaScript Unicode escape sequences unless there is a concrete compatibility requirement.
- If a file shows mojibake, fix the encoding issue at the source instead of propagating the broken text.
- On Windows, prefer UTF-8 terminal sessions when reading or editing files in this repository.
